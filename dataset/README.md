\# Morocco Food Dataset



This folder should contain the Morocco Food Dataset from Kaggle.



\## Download Instructions



1\. Visit: https://www.kaggle.com/datasets/othmanehilal/morocco-food-dataset

2\. Download the dataset

3\. Extract to this folder following this structure:

```

dataset/

├── train/

│   ├── Couscous/

│   ├── Tajine/

│   ├── Harira/

│   └── ...

├── val/

│   ├── Couscous/

│   └── ...

└── test/

&nbsp;   ├── Couscous/

&nbsp;   └── ...

```



\## Dataset Statistics



\- Total images: 5,718

\- Classes: 25

\- Split: 70/15/15 (train/val/test)

\- Image size: 224×224 (preprocessed)
