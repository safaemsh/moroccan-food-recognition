\# Dataset Splits



This folder should contain train/val/test splits of the Morocco Food Dataset.



\## Structure

```

splits/

├── train/

│   ├── Couscous/

│   ├── Tajine/

│   └── ...

├── val/

│   └── ...

└── test/

&nbsp;   └── ...

```



Download the dataset from Kaggle and run the split script to generate this structure.
